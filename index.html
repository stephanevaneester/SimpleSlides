SimpleSlides 5.1

A single HTML document using CSS and JavaScript to make a Powerpoint-like presentation

(Made with love) Forked, by (Jenn Schiffer)Stephane Van Eester.

Get it on Github: https://github.com/stephanevaneester/SimpleSlides

Features

  • Your slides are a web page, so all slide content is HTML.
  • Each parent <div> in the #simpleslides element is an individual slide.
  • Completely customizable style-wise - just add to existing CSS.
  • Click navigation, keyboard shortcuts (← / →)
  • Slide counter - shows what slide out of how many you're on.
  • Looks great on the iPad, which is what I usually present these things on.
  • Heavily documented - read my inline comments and learn how it was made and how to edit.
  • add #slide=[enter slide number here] to the end of your presentation URL to go to that slide!
